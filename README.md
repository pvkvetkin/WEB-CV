# WEB-CV

## Pretty common web page for a CV.
### Technology stack: Nest.js, NodeJS, PostgreSQL, Prisma ORM, Supertokens (OAuth 2), bootstrap.
### This project was deployed on [Heroku](https://web-sem6.herokuapp.com/).
